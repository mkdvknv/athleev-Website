# Athleev Fitness Store

Athleev is a modern fitness supplement eCommerce website built using React.js.  
The platform provides a clean UI/UX experience for users to browse, search, and purchase fitness supplements online.

## Features

- Modern responsive design
- Product listing page
- Product details page
- Add to cart functionality
- Search products
- Category filtering
- Secure checkout flow
- Mobile-friendly UI
- Fast performance

## Tech Stack

- React.js
- JavaScript
- Tailwind CSS
- Vite
- React Router
- Context API

## Products

- Whey Protein
- Isolate Protein
- Mass Gainer
- Creatine
- BCAA

- Multivitamins
- Omega 3

- And more
